Dashboard de performance commerciale (Power BI)

Description : Ce projet analyse la performance commerciale d’une entreprise fictive en comparant les données réelles (Actual), les prévisions (Forecast) et le budget (Budget).

Objectif : Comprendre la performance des ventes et identifier les principaux leviers de croissance.

Insights clés

  • Le chiffre d’affaires est principalement généré par les catégories Home Appliances et Computers.
  • Les ventes réelles dépassent légèrement les prévisions.
  • La croissance est stable dans le temps, indiquant un marché relativement mature.

Outils : Power BI, Data Modeling, Data Visualization

Répartition du chiffre d’affaires par catégorie produit

Description : Ce dashboard fournit une vue globale du chiffre d’affaires total ainsi que sa répartition par catégorie de produits.

Objectif : Identifier les catégories qui contribuent le plus au chiffre d’affaires.

Insights clés

  • Le chiffre d’affaires est concentré sur quelques catégories dominantes.
  • Certaines catégories sont sous-performantes, ce qui crée une opportunité d’optimisation.
  • Risque de dépendance à un nombre limité de catégories.

Analyse de l’évolution mensuelle des ventes

Description : Ce dashboard analyse l’évolution des ventes au fil des mois afin d’identifier les tendances et effets saisonniers.

Objectif : Suivre la performance dans le temps et détecter les périodes de forte ou faible activité.

Insights clés

  • Les ventes augmentent progressivement sur l’année.
